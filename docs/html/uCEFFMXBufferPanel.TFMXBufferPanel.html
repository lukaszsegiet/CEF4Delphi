Unit

uCEFFMXBufferPanel

Declaration

type TFMXBufferPanel = class(TControl)

Description

TBufferPanel is used by FMX applications with browsers in OSR mode to draw the browser contents. See the FMXExternalPumpBrowser demo for more details.
